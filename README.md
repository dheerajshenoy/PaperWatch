# PaperWatch

PaperWatch is an open-source tool designed to help researchers and academics keep track of the latest papers in their field of interest. It provides a simple interface to monitor new publications from various sources, allowing users to stay updated with minimal effort.

## Installation

To install PaperWatch, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/dheerajshenoy/PaperWatch.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PaperWatch
   ```
3. Use pip install (there's a pyproject.toml):
   ```bash
   pip install .
   ```
4. Run the application:
   ```bash
    paperwatch
    ```
## Usage

After installation, you can start using PaperWatch by running the command `paperwatch` in your terminal. This will launch the application, where you can set up your preferences and start tracking papers.
